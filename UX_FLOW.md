# place — UX Flow

## 사용자 여정

```
1. 앱 실행
   └→ 빈 아이소메트릭 방 + "JSON 붙여넣기" 버튼 표시

2. JSON 입력
   └→ 붙여넣기 버튼 클릭 (또는 Ctrl+V / 모바일 붙여넣기)
   └→ furniture_sizes.json 파싱
   └→ 성공: 가구 목록이 사이드 패널에 표시 + 기존 position 있으면 방에 자동 배치
   └→ 실패: 에러 토스트 ("잘못된 JSON 형식입니다")

3. 가구 배치
   └→ 사이드 패널에서 가구 드래그 → 방 바닥에 드롭
   └→ 0.5m 그리드 스냅
   └→ 이미 배치된 가구: 방 안에서 드래그로 이동

4. 회전
   └→ 배치된 가구 탭 → 90도씩 회전 (0 → 90 → 180 → 270 → 0)

5. 충돌 확인
   └→ 겹침 발생: 빨간 테두리 + 경고 아이콘
   └→ 방 밖 드래그: 스냅백 (원위치 복귀)

6. 결과 내보내기
   └→ "JSON 복사" 버튼 클릭
   └→ placement_result.json 클립보드 복사
   └→ 성공 토스트 ("클립보드에 복사되었습니다")
```

### 인터랙션
| 동작 | 결과 |
|------|------|
| 사이드/하단 패널 가구 드래그 → 방 | 가구 배치 (0.5m 스냅) |
| 방 안 가구 드래그 | 위치 이동 (0.5m 스냅) |
| 가구 탭 | 90도 회전 |
| 가구 겹침 | 빨간 테두리 경고 |
| 붙여넣기 버튼 | JSON 입력 다이얼로그 |
| JSON 복사 버튼 | placement_result.json 클립보드 복사 |
